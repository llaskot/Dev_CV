# Dev_CV (React + Vite)

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Deployment to GitHub Pages

The project is already configured for automatic deployment using the `gh-pages` package. The live site is available at: https://llaskot.github.io/Dev_CV/

To publish new changes, run the following commands in your terminal:

1. **Build the project:**

   npm run build

   *This command creates an optimized production build of your site in the `dist` folder.*

2. **Deploy the site:**

   npm run deploy

   *This command pushes the contents of the `dist` folder to the `gh-pages` branch of your repository. Your changes will be live on the site within a few minutes.*
